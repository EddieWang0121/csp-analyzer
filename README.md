# csp-analyzer

Analyze the Content-Security-Policy of a given URL.

## Install

```
git clone https://github.com/gwen001/csp-analyzer
cd csp-analyzer
pip3 install -r requirements.txt
```

## Usage

```
$ python3 csp-analyzer.py <url> [<cookies>]
```
